# SplashScreenDemo

## 中文

### 简介

这是一个基于SplashScreen的示例项目，包含：

```
1、基础(Theme.SplashScreen+Theme.SplashScreen.IconBackground)
2、全屏Lottie
3、全屏Video(基于VideoView)
4、全屏Video(基于ExpoPlayer)
```
